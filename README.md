# Flutter Journey

A collection of Flutter projects built while learning and exploring the Flutter framework. Each folder contains a standalone project with its own README, source code, and setup instructions.

## Projects

| # | Project | Description | Key Packages |
|---|---------|-------------|--------------|
| 1 | [Expense Tracker](./expense_tracker) | Track income & expenses with interactive charts and local storage | Hive, fl_chart, uuid |
| 2 | [Chat App with Firebase](./chat_app_firebase) | Real-time messaging with read receipts, image sharing, and push notifications | Firebase Auth, Firestore, FCM, Firebase Storage |

## About

This repo documents my Flutter learning path — from simple UIs to full-featured apps with state management, local storage, and data visualization. Each project is built to be production-ready, not just a demo.

## Getting Started

Each project is independent. Navigate into any project folder and follow its README for setup instructions.

```bash
cd <project_folder>
flutter pub get
flutter run
```
